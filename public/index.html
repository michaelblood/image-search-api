mb-img.herokuapp.com image search abstraction layer

Example usage:


http://mb-img.herokuapp.com/api/search?q=Karnivool&pretty=pretty

This is the main API endpoint. The above example will return 10 images related to Karnivool, an awesome Australian progressive rock band, in JSON format. The pretty parameter will make the output look good in a browser. In this example, you can see that the q parameter is the search term.


Available API parameters:

Parameter Description Example Required
q the term to be searched q=imogen heap Required
offset a number corresponding to the number of results to skip (for pagination) offset=42 Optional
pretty "Prettify" the output. This is for use with the browser. pretty=pretty Optional

Getting recent searches:

You can also get the 10 latest search terms here:

http://mb-img.herokuapp.com/api/search/latest

The pretty parameter also works here., for example http://mb-img.herokuapp.com/api/search?pretty=pretty

Built using Express, mongoDB, and Google CSE.
